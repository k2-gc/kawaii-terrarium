## Usage

### Via Command Palette
1. Open Command Palette (`Cmd+Shift+P` on Mac, `Ctrl+Shift+P` on Windows/Linux)
2. Type "Kawaii Terrarium"
3. Select:
   - **Summon Mofu on Secondary Sidebar** - Show Mofus in the right sidebar

## Installation

### Option 1
1. Open VS Code
2. Go to Extensions (`Cmd+Shift+X` / `Ctrl+Shift+X`)
3. Search for "Kawaii Terrarium"
4. Click Install

### Option 2
Install from [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=k2-gc.kawaii-terrarium)
